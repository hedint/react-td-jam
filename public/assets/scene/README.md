# Scene Assets

Decorative non-gameplay background layers for Phase 0.5.

Do not bake the path, slots, gate/checkpoint, or Великий Куб into static scene assets.

Use the manifest `scene.*` keys for runtime loading. Production background layers should replace placeholder assets without changing gameplay geometry.
