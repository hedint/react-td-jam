# React TD — Сеттинг и фикшн

> **Рабочий тайтл:** *React TD* (мех-пун: «React» = реакции). Флавор-подзаголовок: *«Последний Батч»*.
> **Это фикшн-слой.** Механика, числа, граф реакций и баланс живут в [`design.md`](design.md) и
> **от сеттинга не зависят** (см. design.md §16). Здесь — мир, лексика и арт-направление поверх той же механики.
> **Статус имён:** дисплей-имена контрапций/реакций/тварей — рабочие, финалятся при полишинге текста.

---

## 1. Логлайн

**Гремлинская твердыня в осаде. В сердце — Великий Перегонный Куб, варящий легендарный «Батч».**
Орда нечисти кружит по осадной галерее, чтобы вскрыть Куб. Гремлины не машут топорами — они
обвешивают кольцо стихийными контрапциями (вода, нефть, искра, жар, холод), которые **сами по себе
бесполезны**, а в связке перемалывают осаду. Грязно, чадно, с чёрным юморком.

Якорь питча («собери химическую машину войны») выживает буквально: гремлины — маньяки
переусложнённой инженерии ловушек, и **2+2=5** здесь = переусложнённая комбо-контрапция.

---

## 2. Народ / POV

- **Кто ты:** прораб осады.
- **Народ:** **гремлины-тинкеры** — работящие, жадные до Батча, помешанные на машинах. Это **НЕ дворфы**
  (DF-вайб берём как референс грязно-индустриальной комедии, но народ свой).
- **Технологии:** джури-риг, на соплях и гениальности. **Электричество — родная гордость гремлинов**
  (пойманная молния, Лейденовы катушки, разрядники) — поэтому «Искра» как эмиттер ничему не противоречит.

---

## 3. Место

- **Твердыня** вырублена в скале над **магма-жилой** — она же греет Куб и питает контрапции «Жар».
- **Осадная галерея** — спиральное кольцо-проход, опоясывающее центральную пещеру. Единственный подход
  к Кубу: враги обязаны протащиться вдоль всех контрапций до **ворот Куба**.
  → это и есть кольцевой путь design.md §7; **ворота = checkpoint = точка урона по Core HP**.
- **Великий Перегонный Куб** — гигантский клёпаный медный аппарат в центре. Варит **«Батч»**.
  - **Уровень Батча растёт на экране** — встроенный idle-прогресс «на экране всегда что-то растёт» (design.md §11).
  - **Core HP = целостность Куба.** Поражение = «Батч пролит».

---

## 4. Тон и арт-направление

- **Тон:** грязно-работящая чёрная комедия. Не ярко-мими, не gritty-дарк — копоть, чад, искры, deadpan.
- **Арт (программерский + генеративный, design.md §2/§13):** чанковые тёмные силуэты, копоть, магма-оранжевое
  свечение ядра, искры/пар частицами. Красота вторична — несёт **VFX реакций**.
- **Читаемость (design.md §11):** наземные реакции на полу, воздушные над клеткой (разная высота);
  максимум одна наземная + одна воздушная на клетку. **Цвет дублируется формой/паттерном** (дальтонизм).

## 5. Эмиттеры — контрапции гремлинов

Эффект буквальный (как в design.md §6), меняется только обёртка имени.

| Эмиттер | Контрапция | Флавор |
|---|---|---|
| **Вода** | Водомёт / насос-цистерна | заливает клетки → водные пулы |
| **Нефть** | Маслонасос / бочка смолы | горючие пулы, по ним ползёт огонь |
| **Искра** | Разрядник / Лейденова катушка | «пойманная молния» — гордость гремлинов |
| **Жар** | Магмовый кран / горелка | тап магма-жилы (она же греет Куб) |
| **Холод** | Морозный мех / криокран | мехи стужи |

## 6. Реакции — дисплей-имена

Рецепты, тиры, слои и роли — без изменений (design.md §6). Здесь только как они называются на экране.

**T1 (вещество + энергия):**
- Вода + Искра → **Электролужа** *(земля)*
- Вода + Жар → **Пар** *(воздух)*
- Вода + Холод → **Наледь** *(земля)*
- Нефть + Жар → **Пожар** *(земля)*
- *(P1)* Нефть + Искра → **Волатил** · *(P1)* Нефть + Холод → **Смола**

**T2 (T1 + соседняя энергия/результат):**
- Пар + Искра → **Грозовое облако** *(воздух)*
- Пожар + Пар → **Огненный вихрь** *(воздух)*
- Наледь + Жар → **Термораскол** *(земля, бёрст по brittle)*
- *(P1)* Волатил + Пожар → **Взрыв**

**T3 (два T2 рядом):**
- Грозовое облако + Огненный вихрь → **Огненный Шторм** («Пекло») *(воздух, чистка экрана)*

## 7. Враги — сборная нечисть

Маппинг 7 архетипов design.md §9.1. Резисты ложатся на природу твари; телеграфятся иконкой заранее
и приходят только когда у игрока уже есть альтернатива (design.md §9.2).

| Архетип | Тварь | Что проверяет | Резист-флавор |
|---|---|---|---|
| Грунт | Гоблин-голодранец | базовое мясо для реакций | — |
| Сварм | Рой пещерных кусак / крыс | AoE / покрытие | — |
| Бегун | Гоблин на варге | контроль (иначе пробежит сквозь DoT) | — |
| Танк | Тролль / огр | стабильный высокий DoT | — |
| Летун | Стая нетопырей / гарпия | **воздушные** реакции | бьётся только воздухом |
| Insulated | **Глиняный голем / грязевик** | альтернатива электро | электро уходит в землю |
| Flameproof | **Магма-выползень / саламандра** | альтернатива огню | рождён в магме, огонь бесполезен |
| *(P1)* Carrier | Гнилостный носитель | — | плодит личинок при смерти |

## 8. Босс — «Бочкоед»

**Глубинный титан-зверь** из нижних пещер, жаждет выпить Батч. Замена «Reactor Leech». Ползёт по кольцу
**3 круга** (механика — design.md §9.4).

- Каждый круг: −урон Кубу, +скорость, и **резист к самому частому типу реакции прошлого круга**
  (идиосинкразия титан-зверя — «у каждого свои уязвимости»). **Правило телеграфится ДО боя**, чтобы игрок
  собрал разнотипную трассу на сетапе.
- **Reaction Break:** получил за круг урон от **3 разных** реакций → «захлебнулся» → Vulnerable 5с, ×2 урон.
  Проверяет качество всей трассы, а не одну точку DPS.
- **Победа** — Бочкоед иссушён. **Поражение** — Core HP 0 (Батч пролит).

---

## 9. Три «вау» (design.md §12) в фикшне

1. **~10 сек:** 2 водомёта + разрядник → 2-клеточная **Электролужа** → первая гоблинская шваль тает.
   *«Две бесполезные железяки = мясорубка».*
2. **Середина:** Вода → Пар → Грозовое облако встык → воздушный шторм выкашивает рой нетопырей и гарпий.
   *«Я собрал машину».*
3. **Кульминация:** Грозовое облако + Огненный вихрь рядом → **Огненный Шторм**, чистка экрана; либо
   Reaction Break на Бочкоеде → Vulnerable → бёрст. `×N` взлетает. *«Сложность окупилась».*
